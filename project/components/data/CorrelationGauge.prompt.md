Semicircular gauge for the **correlation premium** = implied − realized correlation. The intellectual centerpiece of step 5. Always frame the read as "un signal à analyser", never a recommendation.

```jsx
<CorrelationGauge implied={0.61} realized={0.48} size={240} />
```

Premium ≥ +6 pts → "Favorable" (teal), 0..6 → "Neutre" (amber), < 0 → "Défavorable" (red). Inputs are decimals 0..1.
